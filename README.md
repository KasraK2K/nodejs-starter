# STARTER CODE

## Development Mode

To run project as development use this command:

```bash
npm run start:dev
```

If you wish to run development with pm2 on the server use this command:

```bash
npm run pm2:dev
```

---

## Production Mode

To run project as production use this command:

```bash
npm run start
```

If you wish to run development with pm2 on the server use this command:

```bash
npm run pm2
```

---

## Build

For build the project you can use this two command:

```bash
npm run build
npm run build:dev // watch mode
```

---

## Test

For test the code/project use this command:

```bash
npm run test
npm run test:all
```
